# EcoTrack 🌱

EcoTrack is an AI-powered Carbon Footprint Awareness Platform that helps users estimate their environmental impact based on transportation, electricity consumption, and dietary habits.

## Features

- Carbon footprint calculator
- Sustainability score analysis
- AI-powered environmental suggestions
- Responsive dashboard
- User-friendly interface

## Tech Stack

- Firebase Studio
- Next.js
- TypeScript
- Tailwind CSS
- Gemini AI
- Vercel

## Live Demo

https://studio-ten-henna.vercel.app

## Problem Statement

Many people are unaware of how their daily activities contribute to carbon emissions. EcoTrack provides an easy way to measure and understand environmental impact while offering AI-driven suggestions for sustainable living.
