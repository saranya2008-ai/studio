# **App Name**: EcoTrack

## Core Features:

- Sustainability Intake Form: Comprehensive input interface for capturing transport modality, distance, kilowatt-hours, and dietary profiles.
- Dynamic Carbon Engine: Real-time calculation of carbon equivalents (CO2e) based on user inputs using validated emission factors.
- Impact Analytics Dashboard: A responsive dashboard utilizing card-based components to visualize footprint scores across various lifestyle categories.
- AI Emission Reduction Tool: Gemini-powered personalized recommendation engine that uses data-driven reasoning to suggest high-impact behavioral changes.
- Progress Summary: Interactive visual reports and modern card layouts displaying environmental savings milestones.

## Style Guidelines:

- Primary Color: Vibrant Mint Green (#34d399) reflecting modern ecological tech.
- Background Color: Deep Charcoal Teal (#0f1715), providing a high-contrast dark scheme for a premium digital experience.
- Accent Color: Azure Blue (#38bdf8), used for highlights and specific carbon data callouts.
- Headline Font: 'Space Grotesk' (sans-serif) to convey a tech-forward, scientific feel.
- Body Font: 'Inter' (sans-serif) for high readability across the mobile-first dashboard.
- A bento-box style card layout that responds fluidly to different viewport sizes.
- Smooth, subtle CSS transitions for carbon score increments and loading states.
